# 💳 Payment Dashboard Website
DEMO: https://marwaas3d.github.io/modabr/https://marwaas3d.github.io/modabr/

A modern and responsive payment dashboard built with Angular, featuring clean UI, smooth animations, and well-structured components.

---

## 🚀 Live Demo



---

## 📌 Features

* 💳 Manage payment methods (cards UI)
* 📱 Fully responsive design
* 🎨 Modern UI with gradients & glass effect
* ⚡ Smooth animations and transitions
* 🧩 Component-based architecture (Angular)
* 🔍 Clean and reusable CSS structure

---

## 🛠️ Tech Stack

* **Framework:** Angular
* **Languages:** HTML, SCSS, TypeScript
* **Styling:** SCSS (Flexbox + Grid)
* **Icons:** Font Awesome

---


## ⚙️ Installation

```bash
# Clone the repo
git clone git@github.com:marwaas3d/modabr.git

# Go to project folder
cd project-name

# Install dependencies
npm install

# Run the project
ng serve
```

Then open:

```
http://localhost:4200
```

---

## 🎨 UI Highlights

* Glassmorphism effect using:

```css
backdrop-filter: blur(30px);
```

* Gradient buttons:

```css
background: linear-gradient(to right, #f2ca6d, #f0d8a1);
```
